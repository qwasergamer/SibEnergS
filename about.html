Суть сибирского решения

СибЭнергоСети — модуль для автоматической синхронизации оргструктуры из 1С с группами безопасности AD и Samba DC, созданный для обеспечения объектов электроэнергией, размещения опор ЛЭП и управления сетями с трансформаторной мощностью 145 ГВт, 200 тысячами подстанций и 800 тысячами км линий в Сибири.

Процесс в сибирских условиях

1. Получение данных из 1С удаленных филиалов
2. Сравнение с группами AD/Samba в тайге
3. Автосоздание групп для сибирских подразделений
4. Назначение прав с учетом ролей в энергетике
5. Управление доступом для полевых бригад
6. Журнал изменений для регионального аудита
